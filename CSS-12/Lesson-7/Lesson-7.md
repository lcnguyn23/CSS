# CSS12

## Lesson 7 - Setting backgrounds & gradients

### Background color

```css
div {
  background-color: #555;
}
```

### Background image
```css
div {
  background-image: url("example.png");
}
```

#### Background repeat

```css
div {
  background-image: url("example.png");
  background-repeat: no-repeat; /*repeat, repeat-x, repeat-y*/
}
```

#### Background position
```css
div {
  background-image: url("example.png");
  background-position: 20px 10px;
  background-repeat: no-repeat;
}
```
