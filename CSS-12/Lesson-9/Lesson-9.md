# CSS12

## Lesson 9 - Adding media

### Adding images

```html
<img src="example.jpg" alt="example image">
```

### Sizing images

```css
img {
  height: 200px;
  width: 200px;
}
```

### Adding audio
```html
<audio src="jazz.mp3"></audio>
```

### Adding video
```html
<video src="earth.mp4" controls></video>
```

### Adding inline frames

```html
<iframe src="https://www.google.com/maps/embed?..."></iframe>
```
