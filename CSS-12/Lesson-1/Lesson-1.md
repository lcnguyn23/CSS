# CSS12

## Lesson 1 - Building your first web page

### HTML

#### Elements
`<a>`
`<div>`
`<p>`
`<span>`
...

#### Tags
`<a></a>`
`<div></div>`
`<p></p>`
`<span></span>`
...
`<>`: opening tag
`</>`: closing tag

**Self-closing tags**:
`<br>`
`<img>`
`<meta>`
`<link>`
...

#### Attributes
>`<a href="www.example.com">Example</a>`
- Attribute: `href="www.example.com"`

#### HTML Structure
```sh
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="utf-8">
  <title>Hello world</title>
</head>
<body>
  <h1>Hello word</h1>
  <p>This is a web page.</p>
</body>
</html>
```
